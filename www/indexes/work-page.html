THE CLIENT
APCD holds an annual congress for members and clients with more than 50 thousand people in attendance. With the arrival of the pandemic, it was necessary to have a custom applications hosted in the cloud in order to hold the event.
THE SOLUTION
Lymtech helped in the development of a web an mobile application so that APCD could hold the congress online. This implied the need to involve severall technologies such as Azure Cloud, Python, SAP and Vimeo as Live Streaming.
THE RESULT
The event could be done in Pearson, but had more than 2000 online attendees who were able to enjoy and interact on the congress. In addition, a backoffice was developed with a bootchar, organization of stands, agenda, registration to the talks, key notes, among others.

THE CLIENT

Banco Safra is one of the ten largest banks in Brazil, part of the Safra group.We collaborated in the implementation of a fraud analysis software. ​

The challenge we had was to integrate all their transactional and master data to this tool. 

THE SOLUTION

We created an integration system (middelware) to migrate and transform the information from one system to another. ​

We found a very high complexity when linking entities with the financial movements of each account, so we developed a parameterized matrix to map all the movements that the bank needed.

THE RESULT

We work under strict security norms given that we comply with the client's standards and requirements.

The software developed by Lymtech allowed our client to generate a faster and more accurate dynamic in terms of fraud analysis, thus minimizing operating times.
