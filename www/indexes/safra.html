Banco Safra

Developing platforms and mobile apps to help people live healthier lives

About the Client

Banco Safra is one of the ten largest banks in Brazil, part of the Safra group.We collaborated in the implementation of a fraud analysis software. ​ The challenge we had was to integrate all their transactional and master data to this tool.

The Solution

We created an integration system (middelware) to migrate and transform the information from one system to another. ​ We found a very high complexity when linking entities with the financial movements of each account, so we developed a parameterized matrix to map all the movements that the bank needed.

The Result

We work under strict security norms given that we comply with the client's standards and requirements. The software developed by Lymtech allowed our client to generate a faster and more accurate dynamic in terms of fraud analysis, thus minimizing operating times.
